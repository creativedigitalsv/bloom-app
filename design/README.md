# Design

## Klasor Yapisi
- `wireframes/` - Figma exportlari, el cizimleri
- `ui-kit/` - Renk paleti, tipografi, ikonlar
- `prototypes/` - Interaktif prototip linkleri

## Marka Kimligi
- **Ana Renk:** Soft pembe/mor tonlari (hamilelik = sicaklik)
- **Font:** Modern, okunakli sans-serif
- **Stil:** Minimal, sicak, guven veren
- **Ikon Stili:** Outline, rounded corners
