# Product Department

## Sorumluluk Alani
Urun yol haritasi, PRDler, kullanici arastirmasi, UX.

## Klasor Yapisi
- `prd/` - Product Requirements Documents
- `user-research/` - Kullanici gorusmeleri, anketler
- `roadmap/` - Sprint planlari, milestoneler

## Oncelikli Gorevler
1. MVP PRD yazimi (8-12 hafta scope)
2. Kullanici persona tanimlamalari
3. Feature priority matrix (MoSCoW)
4. Sprint 1 planlama
5. Wireframe review ve onay

## MVP Ozellikler (Oncelik Sirasi)
1. Hamilelik takip takvimi (hafta/trimester)
2. Partner sync (cift hesabi)
3. Gunluk saglik kontrolu
4. Bilgi kutuphanesi (haftalik gelisim)
5. Push notification sistemi
