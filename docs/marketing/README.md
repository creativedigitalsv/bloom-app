# Marketing Department

## Sorumluluk Alani
Marka stratejisi, dijital pazarlama, icerik uretimi, SEO.

## Klasor Yapisi
- `brand/` - Logo, renk paleti, marka kilavuzu
- `campaigns/` - Kampanya planlari ve materyalleri
- `research/` - Pazar arastirmasi, rakip analizi
- `content/` - Blog, sosyal medya, newsletter icerikleri
- `seo/` - SEO stratejisi, anahtar kelime arastirmasi

## Oncelikli Gorevler
1. Brand identity olusturma (Bloom = cicek acma metaforu)
2. Isvec pazari rakip analizi (Natural Cycles, Kry, Min Doktor)
3. Pre-launch landing page
4. Sosyal medya stratejisi (Instagram, TikTok)
5. Brandwatch 2026 trend aksiyonlari (bkz: marketing brief)
