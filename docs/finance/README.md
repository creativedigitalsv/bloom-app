# Finance Department

## Sorumluluk Alani
Butce yonetimi, fon basvurulari, mali raporlama.

## Klasor Yapisi
- `budgets/` - Aylik/yillik butce planlari
- `funding/` - Vinnova, ALMI, EU fonlari basvurulari
- `reports/` - Mali raporlar, P&L, cash flow
- `invoices/` - Fatura sablon ve kayitlari

## Oncelikli Gorevler
1. MVP butce plani (8-12 hafta)
2. Vinnova Innovativa Startups 2025 basvuru hazirlik
3. ALMI mikroloan arastirma
4. 18 aylik financial projection
5. Burn rate hesaplama
