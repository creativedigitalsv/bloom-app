# Analytics Department

## Sorumluluk Alani
Veri analizi, KPI takibi, kullanici davranisi, raporlama.

## Klasor Yapisi
- `reports/` - Haftalik/aylik performans raporlari
- `dashboards/` - Dashboard tanimlamalari ve konfigurasyonlari

## Oncelikli Gorevler
1. KPI framework tanimlama
2. Analytics altyapi secimi (Mixpanel/Amplitude/PostHog)
3. Kullanici acquisition funnel tanimlama
4. A/B test framework
5. Haftalik rapor sablonu

## Temel KPIlar
- DAU/MAU ratio
- Retention (D1, D7, D30)
- Partner invite conversion rate
- Feature adoption rate
- NPS score
